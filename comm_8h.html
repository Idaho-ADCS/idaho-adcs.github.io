typedef int8_t fixed5_3_t
 Defines a fixed-point data type that is one byte wide. Five bits are reserved for the integer part and 3 bits are reserved for the fraction part. While in fixed-point form, the data cannot be read - it must be converted back to floating-point first. Functions that convert float to fixed and back are declared below.
